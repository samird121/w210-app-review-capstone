Need to do once:
1. Download and install node.js and npm to your computer
2. Install the scraper on the command line of your terminal with `npm install google-play-scraper` (without the `s)

Need to do every day:
1. In terminal, navigate to the folder with scrape.js file in it
2. Copy paste the lines in the appropriate scrape_batch .txt file into your command window, which will slowly download reviews and append them to the appriorate scraped_reviews .txt file
3. Push to github repo
